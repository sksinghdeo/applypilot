# Roadmap

## v1.1 — Reliability

- encrypted IndexedDB profile storage;
- improved Greenhouse, Lever, Workday, LinkedIn, and Indeed extraction fixtures;
- duplicate-application warnings;
- answer-template editor;
- accessibility audit and keyboard-flow improvements.

## v1.2 — Review workflow

- per-field evidence links back to source profile facts;
- stronger uncertainty scoring;
- application checklist and follow-up reminders;
- export/import of a fully encrypted workspace.

## v2.0 — Extensible platform

- provider-side structured-output validation;
- extension adapter API for additional ATS portals;
- optional self-hosted database with authentication;
- team-safe deployment documentation.

The project will retain explicit human approval for sensitive answers and final submission.
