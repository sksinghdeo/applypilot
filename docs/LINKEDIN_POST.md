# LinkedIn launch package

## Recommended post

I got tired of entering the same information into every job portal—while still needing to carefully review the questions that actually matter.

So I built **ApplyPilot**, a privacy-first, human-in-the-loop copilot for job applications.

ApplyPilot can:

→ build a reusable candidate profile and resume library  
→ extract the title, company, location, work mode, and compensation from a job page  
→ analyze fit, surface gaps, and detect common eligibility blockers  
→ route the strongest resume for the opportunity  
→ prepare truthful, reviewable application answers  
→ fill only routine contact fields through a Chrome extension  
→ keep sensitive answers and final submission under the candidate’s control

The app runs locally by default and works without an API key. Optional OpenAI, Anthropic, and Gemini adapters are included for richer analysis.

This project brought together product design, structured data, document parsing, browser automation, local heuristics, model-provider adapters, and safety guardrails around a problem I have experienced firsthand.

**AI handles the repetition. The candidate keeps control.**
